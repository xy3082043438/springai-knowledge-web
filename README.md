# SpringAI Knowledge Web (企业级知识库管理台)

> 基于 **Vue 3.5**、**Vite 7.2** 和 **Element Plus** 打造的现代化企业知识库与问答管理前端平台。
> 秉承“**高密度、一屏流**”的设计理念，为 SpringAI Knowledge Server 提供极致流畅的管理交互与 AI 对话体验。

---

## ✨ 核心特性

- 🎨 **极致打磨的 UI/UX**: 
  - 采用现代化审美设计（玻璃拟态、光顺动效），全局高度统一了日期时间格式规范与表格交互偏好。
  - **组件级优化**: 侧边栏与菜单具备丝滑的过渡动画，全局支持头像（Avatar）聚合展示，提升系统温度。
- 📊 **集约化全维仪表盘**:
  - **密度优先策略**: 精心设计的极简高密度布局，**无需滚动即可一屏全览**所有关键运行指标、知识分布与问答趋势。
  - **深度数据透视**: 集成 **ECharts 5.6** 与 WordCloud，直观展示系统负载与知识库热点词云。
- 💬 **沉浸式智能问答 (Smart QA)**:
  - **流式对话**: 仿主流 AI 的聊天界面，支持 SSE（Server-Sent Events）毫秒级流式响应。
  - **智能引导**: 自动读取文档相关的 **AI 推荐问题**，引导用户深度挖掘知识库价值。
  - **精密溯源**: 自动标注引用文档、段落来源，并支持一键定位至原始分段预览。
- 🛡️ **现代化 RBAC 管理**:
  - **极致权限分配**: 抛弃传统的纵向滚动勾选，采用全新的 **横向压缩式权限面板**，支持基于组的图标化权限分配，高效直观。
  - **深度审计**: 提供操作日志、问答日志、反馈日志面板，并集成后端 **Excel 导出库**，实现数据离线分析。
- 📁 **知识库中台**:
  - 文档上传、异步解析状态深度追踪。
  - **多维预览**: 支持原生的 PDF 渲染、Markdown 检视及切片详情预览。
  - **细粒度权限**: 支持对单个文档或分类设置角色级访问隔离。

---

## 🛠 技术架构

- **核心层**: Vue `^3.5.25`, TypeScript `^5.7.2`
- **构建工具**: Vite `^7.2.4` (极致的 HMR 体验)
- **UI 框架**: Element Plus `^2.13.0`
- **状态/路由**: Pinia `^3.0.4`, Vue Router `^4.6.4`
- **可视化**: ECharts `^5.6.0`, ECharts WordCloud
- **网络**: Axios (高度封装，集成 JWT 拦截与统一错误处理)
- **通用工具集**: 独立的日期与文本工具库 (`utils/date.ts`) 规范全局展示逻辑。

## 📁 项目目录导航

```text
src/
  ├── api/          # 分模块定义的后台接口调用 (auth, business, system)
  ├── components/   # 高复用业务组件 (Dashboard 仪表盘组件群、沉浸式预览器等)
  ├── layout/       # 系统骨架 (Sidebar, Navbar, AvatarHeader)
  ├── router/       # 路由配置与基于角色的鉴权守卫
  ├── store/        # Pinia 状态树 (用户 Token, 全局配置)
  ├── styles/       # 系统全局样式与 Sass 混合 (含透明/模糊动效)
  ├── types/        # 强类型 TypeScript 接口定义
  ├── utils/        # 工具类 (权限判断、Request 封装、日期格式化)
  └── views/        # 功能页面 (单屏看板、智能 Q&A、文档中心、RBAC 系统设置)
```

## 🚀 快速启动

### 环境要求
- **Node.js**: `^20.19.0` 或 `>=22.12.0`

### 1. 安装依赖
```bash
npm install
```

### 2. 本地开发
```bash
npm run dev
```
> 默认运行于: `http://localhost:3000`
> 代理配置: 本地开发请求将自动代理至 `http://localhost:8080/api`。

## 🔑 安全与一致性

- **令牌机制**: 登录后 JWT Token 持久化于 `localStorage`，所有请求自动挂载 `Authorization` Header。
- **一屏流体验**: 通过全局样式约束，确保核心模块在标准 1080P/2K 分页下实现真正的“免滚动”操作，极大提升管理效率。
- **权限闭环**: 前端路由守卫与后端权限校验双重保障，非授权角色无法看到或调用受限资源。
