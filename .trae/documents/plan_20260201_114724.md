## 清空项目源码计划

**目标**：将项目恢复到初始创建状态，只保留配置文件

### 执行步骤：

1. **删除 src 目录下所有文件和子目录**
   - 删除 src/api/ 目录
   - 删除 src/assets/ 目录
   - 删除 src/components/ 目录（保留空目录结构）
   - 删除 src/layouts/ 目录
   - 删除 src/router/ 目录
   - 删除 src/store/ 和 src/stores/ 目录
   - 删除 src/styles/ 目录
   - 删除 src/utils/ 目录
   - 删除 src/views/ 目录
   - 删除 src/App.vue
   - 删除 src/main.ts / main.js
   - 删除 src/env.d.ts
   - 删除 src/styles 目录

2. **保留的文件**：
   - package.json
   - vite.config.ts
   - tsconfig.json / tsconfig.node.json
   - jsconfig.json
   - index.html
   - .gitignore
   - .vscode/ 目录
   - public/ 目录

3. **保留空的 src 目录结构**

### 操作后项目状态：
- 项目将只包含配置文件
- src 目录将为空
- 可以随时重新生成代码
